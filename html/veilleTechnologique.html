Qu'est ce que c'est que ce React ?


React a été créé par Jordan Walke, un ingénieur logiciel travaillant pour Facebook.
La première utilisation de cette bibliothèque date de 2011 (Facebook).
Plus tard en 2012 c'est Instagram qui l'utilisera.
React est une très grosse bibliothèque JavaScript mise à disposition des utilisateurs afin d'aider à la création d'interfaces utilisateur.

Pourquoi utiliser React ?

Avec React il est possible de créér des composants réutilisables !
Par ailleurs React utilise JSX qui permet alors d'utiliser du HTML dans du JavaScript afin d'afficher des sous-composants.
De plus React a évolué et permet depuis 2015 de faire du Native.
Le React Native permet de fournir l'architecture react aux applications natives tel que IOS et Android.
React se démarque grace à :
Sa Simplicité / Son approche native / Sa performance / Sa testabilité.

Septembre 2020

JSX Transfrom

Un navigateur ne comprend pas le JSX.
C'est pourquoi on passe par des compilateurs tel que Babel ou TypeScript.
Mais certains framework tel que React App ou Next.js compile directement le JSX et sont donc bien plus efficace pour faire du React.

Nouveaux avantages :

  • Utiliser JSX sans importer React
  • Sortie compilée légèrement amélioré
  • Facilite les prochaines améliorations
  • Import désormais automatique des fonctions spéciales

Mars 2022

REACT 18.0

Simultanéité : Il s'agit d'un nouveau mécanisme permettant à React de préparer plusieurs versions de votre interface utilisateur en même temps. (multi-version)

Transitions : distinction entre les mises à jour en temps réel.

Suspense : permet de spécifier de manière 'déclarative' l'état de chargement d'une partie de l'arborescence des composants (si elle n'est pas encore prête à être affichée).

Nouvelles API de rendu client et serveur (client DOM / serveur DOM).

Nouveaux crochets : 'useId', 'useTransition', 'startTransition', 'useDeferredValue', 'useSyncExternalStore' et 'useInsertionEffect'.
